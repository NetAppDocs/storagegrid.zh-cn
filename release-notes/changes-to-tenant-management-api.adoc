= StorageGRID 中租户管理 API 的更改
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
StorageGRID 12.1 使用租户管理 API 的版本 4。版本 4 弃用版本 3；但是，仍支持版本 1、2 和 3。


NOTE: 您可以继续在 StorageGRID 中使用已弃用的租户管理 API 版本；但是，在未来的 StorageGRID 版本中，将移除对这些 API 版本的支持。升级到 StorageGRID 12.0 或更高版本后，您可以使用 `PUT /grid/config/management` API 停用已弃用的 API。

要了解更多信息，请访问link:../tenant/understanding-tenant-management-api.html["了解租户管理 API"]。



[[new-api-for-object-replication]]
== 用于对象复制的新 API

您可以使用 `/org/cross-grid-replication/{versionId}/replicate`用于在新配置了跨网格复制的现有存储桶上手动复制对象的 API。
