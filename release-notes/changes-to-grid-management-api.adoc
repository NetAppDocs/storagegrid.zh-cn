= StorageGRID 中网格管理 API 的更改
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
StorageGRID 12.1 使用网格管理 API 第 4 版。版本 4 弃用版本 3；但是，仍支持版本 1、2 和 3。


NOTE: 您可以继续在 StorageGRID 中使用已弃用的管理 API 版本；但是，在未来的 StorageGRID 版本中，将移除对这些 API 版本的支持。升级到 StorageGRID 12.0 或更高版本后，您可以使用 `PUT /grid/config/management` API 停用已弃用的 API。

要了解更多信息，请访问link:../admin/using-grid-management-api.html["使用网格管理 API"]。



[[new-endpoints-for-changing-the-bmc-password-and-lan-ip-settings]]
== 用于更改 BMC 密码和 LAN IP 设置的新端点

已添加以下端点，以允许您link:../harden/hardening-guidelines-for-storagegrid-nodes.html#change-appliance-bmc-password["更改设备 BMC 密码"]。

`PUT /bmc-configuration/root-password`

已添加以下端点，以允许您link:../maintain/changing-nodes-network-configuration.html#change-appliance-node-bmc-lan-ip-settings["更改或查看 BMC LAN IP 设置"]。

`PUT /grid/bmc/{node-id}/lan-ip-config`

`GET /grid/bmc/{node-id}/lan-ip-config`
