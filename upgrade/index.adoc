= 升级 StorageGRID 软件
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
按照以下说明将 StorageGRID 系统升级到新版本。

执行升级时、StorageGRID系统中的所有节点都会升级。


CAUTION: 如果您急需扩展存储以增加容量（如果您的存储几乎已满），请不要开始升级。及时完成升级程序。如果需要恢复，请联系技术支持。
