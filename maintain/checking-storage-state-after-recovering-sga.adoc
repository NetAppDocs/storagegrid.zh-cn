= 恢复设备存储节点后检查存储状态
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
恢复设备存储节点后，您必须验证设备存储节点的所需状态是否设置为联机，并确保每当重新启动存储节点服务器时，此状态默认为联机。

.开始之前
* 您已使用登录到网格管理器link:../admin/web-browser-requirements.html["支持的 Web 浏览器"]。
* 存储节点已恢复，数据恢复已完成。


.步骤
. 选择“*节点*”>“*存储节点*”>“*任务*”。
. 如果“*存储状态*”下拉列表设置为“只读”或“离线”，请选择“*在线*”。
. 选择 * 保存 * 。
