= 客户端请求冲突
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
冲突的客户端请求（例如，两个客户端写入同一密钥）将以 " 最新成功 " 为基础进行解决。

" 最新赢单 " 评估的时间取决于 StorageGRID 系统何时完成给定请求，而不是 S3 客户端何时开始操作。
