= 管理存储节点
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
存储节点可提供磁盘存储容量和服务。管理存储节点需要执行以下操作：

* 管理存储选项
* 了解什么是存储卷水印，以及如何使用水印覆盖来控制存储节点何时变为只读
* 监控和管理用于对象元数据的空间
* 为已存储对象配置全局设置
* 正在应用存储节点配置设置
* 管理完整存储节点
