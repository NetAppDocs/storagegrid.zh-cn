= 警报参考
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
此参考列出了网格管理器中显示的默认警报。建议的操作会显示在您收到的警报消息中。

您可以根据需要创建自定义警报规则，以适合您的系统管理方法。

某些默认警报使用link:commonly-used-prometheus-metrics.html["Prometheus 指标"]。
